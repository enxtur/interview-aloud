title: Security Best Practices
question: What are common security best practices in backend systems?
tags: [security]
difficulty: medium
keywords: [Security, Best Practices, Validation, Sanitization, Hashing, Authentication, Authorization]
---
Always validate and sanitize user input.
Sensitive data should never be stored in plain text.
Passwords should be hashed using strong algorithms.
Authentication and authorization must be enforced on every request.
Secrets should be stored securely and not in source code.
Dependencies should be kept up to date.
