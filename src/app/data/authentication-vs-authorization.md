title: Authentication vs Authorization
question: What is the difference between authentication and authorization?
tags: [security, auth]
difficulty: easy
keywords: [Authentication, Authorization, Security, Authentication vs Authorization, Role, Permission]
---
Authentication is the process of verifying who a user is.
Authorization determines what an authenticated user is allowed to do.
Authentication usually happens before authorization.
Logging in with a username and password is authentication.
Checking permissions or roles is authorization.
Both are required for secure systems.
