title: Horizontal vs Vertical Scaling
question: What is the difference between horizontal and vertical scaling?
tags: [scalability, architecture]
difficulty: medium
keywords: [Horizontal, Vertical, Scaling, Physical Limits, Fault Tolerance, Modern Systems]
---
Vertical scaling means adding more resources to a single machine.
Horizontal scaling means adding more machines to handle load.
Vertical scaling has physical limits.
Horizontal scaling improves fault tolerance.
Modern systems prefer horizontal scaling.
