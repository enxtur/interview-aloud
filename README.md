# Interview Aloud

**Interview Aloud** is a lightweight, browser-based tool for practicing technical interview answers out loud.

- Curated interview questions and concise model answers
- Sentence-by-sentence text-to-speech playback
- Keyboard-friendly, distraction-free practice flow
- No backend, no accounts, no tracking

The goal is simple: help engineers speak clearly and confidently about core technical topics under interview conditions.
